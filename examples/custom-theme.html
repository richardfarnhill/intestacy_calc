About This Example

This example demonstrates how to customize the appearance of the intestacy calculator to match your website's branding. The calculator below uses a purple-themed custom style.

You can customize the calculator by:

  • Adding custom CSS to override the default styles
  • Using the theme option when initializing the widget
  • Adding custom classes to the container
